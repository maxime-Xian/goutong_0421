### Item 11: 光的颗粒感与 X 光原理 (10岁)
**观察**：
- 电筒光在夜空呈现颗粒状，近距离看不到
- 下午 3 点的阳光可以穿透衣服/肉体，看到骨骼投影
- 联想到 X 光透射原理

**洞察**：
```
"是不是有足够的空间给这些光，这些光就呈现出颗粒状？"
"它为什么可以穿透人的衣服、肉体，直接看到骨架？"
```

**意义**：10 岁时已独立发现 X 光的基本原理，展现跨维度类比能力。
