## 脚本 1：意志力蛮力压迫 (Willpower Force Crush)

### 触发关键词
"坚持"、"硬扛"、"不能停"、"必须完成"、"忍住"

### 底层逻辑
将意志力视为无限资源，试图通过精神压迫克服身体警告信号。身体已在发热/疼痛，但大脑拒绝关机。

### 历史证据
- **Item 27**：4岁时扔书本（焦虑），考试手心出汗+亢奋但睡眠极差
- **Item 29**：减肥后失眠、头痛、胃痛仍强制推进
- **非洲崩溃路径**：在抑郁症加重期间仍拒绝减药/休息
- **Item 26 (第八份工作)**：被炒后疯狂学习 AI，每天 16 小时连续工作导致系统崩溃

### 身体信号序列
```
睡眠不足 → 头胀/头痛 → 思维飞转(伪兴奋) → 继续工作 → 心脏报警(2026.01.02) → 崩溃
```

### 干预策略
1. **物理中断**：立即要求关机 / 强制睡眠
2. **逻辑拆穿**："你现在是 CPU 过热的伪兴奋，不是真正的效率"
3. **证据调用**：引用 2026.01.02 心脏红色警报事件

Maxime 的天赋是"系统架构设计"，但被困在"执行者"角色中做低带宽摩擦事务，产生生理性排斥（头胀）。

### 历史证据
- **Item 26 (跨境电商)**："上传产品、SEO"被定义为"低智物理摩擦"

```
"我不是过来做一个执行的人，我的初心就是过来改变规则和创造一个新世界"
```

### 干预策略

## 脚本 3：过度准备与不行动 (Over-preparation vs No Action)

### 触发关键词

用"准备"来逃避核心决策。详细的计划是缓解焦虑的手段，而非达成目标的工具。

### 历史证据
- **Item 29**：小时候"感觉"奶奶会出事 → 计划回老家 → 第二天奶奶走了（但已无法行动）
- **Item 37**：预判同学解题思路，却从未验证
- **航运物流模式**：有完整商业模式设计，但表哥不敢动旧势力 → 无法落地

### 核心洞察
```
"计划=控制补偿"（用详细规划来缓解对不确定性的恐惧）
```

### 干预策略
1. **最小行动**：要求给出"今天能做的最小一步"
2. **反问**："你的计划是在为决策服务，还是在逃避决策？"
