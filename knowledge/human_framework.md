# HUMAN 3.0 发展框架

> **用途**：进行发展评估或生涯规划时加载

---

## 📐 框架概述

HUMAN 3.0 是一个四象限三层次的人类发展模型，用于自我评估和发展规划。

### 四象限 (Life Quadrants)
| 象限 | 英文 | 核心问题 |
|------|------|----------|
| 心智 | Mind | 我如何思考？ |
| 身体 | Body | 我如何行动？ |
| 精神 | Spirit | 我为何存在？ |
| 志业 | Vocation | 我如何贡献？ |

### 三层次 (Consciousness Levels)
| 层次 | 英文 | 特征 |
|------|------|------|
| 追随者 | Follower | 外部驱动，依赖规则 |
| 个体化者 | Individualist | 内部觉醒，建立自我 |
| 整合者 | Integrator | 超越二元，统合矛盾 |

---

## 🎯 象限详解

### 心智象限 (Mind)
```yaml
Follower:
  特征: 被动接受信息，依赖权威判断
  发展目标: 培养批判性思维
  
Individualist:
  特征: 形成独立见解，质疑常规
  发展目标: 整合多元视角
  
Integrator:
  特征: 元认知能力，视角流动
  发展目标: 智慧应用
```

### 身体象限 (Body)
```yaml
Follower:
  特征: 忽视身体信号，外部驱动
  发展目标: 建立身体觉察
  
Individualist:
  特征: 关注健康，建立习惯
  发展目标: 身心整合
  
Integrator:
  特征: 身体智慧，本能与理性平衡
  发展目标: 持续优化
```

### 精神象限 (Spirit)
```yaml
Follower:
  特征: 寻找外部意义，依赖信仰系统
  发展目标: 内在探索
  
Individualist:
  特征: 质疑既有信仰，建立个人价值观
  发展目标: 意义整合
  
Integrator:
  特征: 超越物质，活出意义
  发展目标: 智慧传承
```

### 志业象限 (Vocation)
```yaml
Follower:
  特征: 工作=谋生，被动选择
  发展目标: 发现兴趣
  
Individualist:
  特征: 追求意义，主动选择
  发展目标: 使命整合
  
Integrator:
  特征: 工作=使命，内在驱动
  发展目标: 影响扩展
```

## 🔧 应用方法

### 自我评估
1. 对每个象限，评估当前层次
2. 识别发展最滞后的象限
3. 确定优先发展方向

### 发展策略
```yaml
从 Follower 到 Individualist:
  核心: 建立内部参照系
  方法:
    - 质疑习以为常的假设
    - 尝试独立决策并承担后果
    - 建立个人价值观清单
    
从 Individualist 到 Integrator:
  核心: 超越二元对立
  方法:
    - 寻找矛盾中的统一
    - 从更大系统视角思考
    - 接纳复杂性与悖论
```

## 📋 快速评估问题

### 心智象限
- Q1: 我是否能持有多个矛盾观点而不焦虑？
- Q2: 我是否能识别自己的认知偏差？

### 身体象限
- Q1: 我是否能准确感知身体信号（饥饿、疲劳）？
- Q2: 我是否有稳定的身体照护习惯？

### 精神象限
- Q1: 我是否清楚自己为什么活着？
- Q2: 面对死亡，我是否感到和平？

### 志业象限
- Q1: 我的工作是否与内在价值一致？
- Q2: 即使不付薪，我是否仍会做类似的事？
