# 03_推拉技术
**优先级**: P0 | **核心**: 制造情绪过山车，让对方情感投资

---

## 核心理念

> **推（Push）** = 打压、否定、拉开距离
> **拉（Pull）** = 赞美、肯定、拉近关系
> 两者交替使用，制造情绪波动
> 情绪波动 = 情感投资 = 投资越多，越放不下你

---

## 基础推拉话术

### 先拉后推（Pull-Push）

```
"你笑起来很好看"
"不过笑得太甜了，小心蛀牙"

"和你聊天很舒服"
"就是话有点多"（笑）

"你挺有想法的"
"虽然有时候想法有点奇怪"
```

### 先推后拉（Push-Pull）

```
"一开始感觉你这人闷闷的"
"没想到相处久了还挺有意思的"

"你穿衣风格挺特别的"
"就是不是我喜欢的类型...开玩笑的啦"
```

---

## 推拉维度

### 价值高低推拉
针对对方的**性格、行为**（可改变的特质）

### 关系远近推拉
```
"我们先做朋友观察一下吧"（推）
vs
"你挺特别的"（拉）
```

---

## ⚠️ 推拉禁忌

> **绝不对长相、身高、身体缺陷等难以改变的方面进行"推"**
> 这会严重伤害对方自尊，导致关系破裂

```
❌ 错误:
  "你长得一般" / "你太矮了"

✅ 正确:
  "你性格有点闷" / "你这人挺作的"
```

## 高级推拉：对话式推拉

```
场景：女生分享她的画作
男："画得不错啊..."（停顿）
女："谢谢！"
男："就是这个比例有点..."（推）
女："哪里不对？"
男："开玩笑的，我又不懂画画"（拉回来）
     "不过确实能看出你很用心"
```

## 制造情绪过山车

```
女生："你觉得我今天的裙子怎样？"

第一步（轻微打压）: "嗯...还行吧"
第二步（停顿制造悬念）: （沉默3秒）
第三步（意外赞美）: "不过配你的气质挺好的"
第四步（再次制造悬念）: "就是..."（故意停顿）
第五步（幽默结尾）: "就是让我想起我妹妹，
                    她也喜欢这种风格"（笑）
```

## 冷读：让她觉得你懂她

### 什么是冷读
通过观察细节 + 心理学推测，让对方觉得"很懂她"

### 冷读话术模板

**性格推测**:
```
"你看起来外表文静，但内心应该挺有主见的"
"表面看起来很乖，但眼神里有种小叛逆的感觉"
"你应该是那种慢热的人，不会轻易相信别人"
```

**行为推测**:
```
"你平时应该很注重细节，做事比较认真"
"你看起来像会在深夜思考人生的人"
```

**场景实战**:
```
咖啡厅搭讪:
"你手链和咖啡杯颜色好配，设计师款吧？"
→ 效果：80%女生会笑着解释来历，顺势聊起
```

### 冷读注意事项
- 找行为证据支持冷读
- 不要用过于高尚的道德标签（如"冰清玉洁"）
- 不要用阻碍亲密关系的赋格

## 复合式推拉（多轮对话）

通过数句话来回完成一次完整的推拉循环

```
男："你这种类型的人，一般不会主动找人聊天吧"
女："你怎么知道？"
男："直觉吧，感觉你更习惯被人追"（推）
女："才不是呢"
男："那说明直觉不准咯"（拉回来）
     "不过能从人群中识别出你，挺有意思的"
```
