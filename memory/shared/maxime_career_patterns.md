# 马克西姆原始经历：职业生涯模式

## 职业生涯模式 (8份工作的共同结构)
### 共同模式
```
进入 → 快速理解系统 → 发现结构性问题 → 提出改革方案
→ 遭遇阻力（上级/合伙人/体制）→ 能量消耗 → 离开
```
### 代表性案例
**科特迪瓦（最成功）**：
- 从0→100万$两个月
- 核心：雇Tata（竞争对手员工）快速获取数据；搭建可复制体系
- 关键信念："体系比销售数据重要"
- 崩溃：抑郁症加重→被迫离开
**脑船长（2019重返）**：
- 高压观摩会模式
- 身体警报后继续加班
- 团队冲突+对公司不信任
### 模式标签
- 每份工作都：6个月-2年内离开
- 离开理由：价值观冲突/体制限制/上级不采纳意见
- 共同特点：能快速看清系统问题，但执行层面遇阻
