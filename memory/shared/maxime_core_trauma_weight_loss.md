# 马克西姆原始经历：核心创伤——12-13岁减肥地狱

## 核心创伤：12-13岁减肥地狱（意志力崇拜起源）
### 触发事件
- 被嘲笑"大笨猪"（体重128斤/身高1.5米）
- 核心恐惧激活："我最在意自己聪不聪明"
### 极端方法
- 每天：1200俯卧撑+1200仰卧起起+2小时跑步+打球
- 饮食：早晚各2碗粥，极度饥饿
- 惩罚机制：目标100个只做90个→10个×10=再做100个
### 建立的三个致命信念
1. **意志力崇拜**："只要咬牙坚持，就能战胜一切"
2. **死办法替代灵气**：放弃灵活策略，改用题海/死记硬背
3. **极端自我苛刻**：非满分即失败
### 崩溃 (16岁初三)
- 站着睡着；眼睛翻白；鬼压床频繁
- 成绩：全校第二 → 全班倒数
- 失去重点班/保送资格
### 长期后果
- 胃病（持续至今）
- 近视
- 体质急剧下降（以前很少生病）
- 睡眠不稳定（持续40年未解决）
- "意志力万能"旧脚本植入
