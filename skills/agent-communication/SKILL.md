## 核心模块索引

### M-1 Stage Mapping [阶段映射]
阶段判断 → 匹配策略:
- **破冰期**: 自我引导(70-30) + 电影法则 + 荒谬性
- **暧昧期**: 推拉 + 服从性测试 + 性张力诱导
- **舒适期**: 有利描述(暗度) + 人格赋格 + 冷读
- **危机期**: FRM-3 + VEX-5 + 幽默化解
- **长期关系**: 怀旧效应 + 推拉动态 + 新鲜体验

### M-2 Attraction Architecture [吸引建构器]
- 电影法则: 你是电影,对方是观众
- 70-30法则: 70%聊自己,30%聊对方
- DHA间接展示: 通过故事/细节展示高价值
- 荒谬性态度: 扯淡幽默统一情绪

### M-3 Meta-Scripts [催眠脚本引擎]
- 冷读术: "你看起来外表文静,但眼神里有小叛逆"
- 有利描述(暗度): "你愿意跟我玩这个游戏,说明你对我兴趣不小"
- 观察者效应: 你看到什么样的人,她就会变成什么样的人
- 重复植入: 信息重复足够多次→开始相信

### M-4 Sexual Induction [性张力诱导器]
- 推拉暧昧边界: 赞美+轻微打压
- 吊桥效应: "空调太冷?要不猜猜凶手是谁,猜错罚靠我肩膀取暖"
- 身体语言暗示: 通过语言制造接近的想象画面
- 制造独特性: "你是第一个能理解我这个想法的人"

### M-5 Frame Mastering [框架主导引擎]
**FRM-3 (情绪主导型回应):**
- 废物测试: 不直接回答 → 框架反转
  * "你是不是很会撩女生?" → "你不是正在和我探讨这个问题吗?"
  * "你这么优秀为什么单身?" → "在等能发现我隐藏技能的人"
- 误解/冲突: 同意并夸大 → 荒谬化
- 关系冲突: 跳出问题 → 重塑更高维度框架

**VEX-5 (高价值展示后自动降火):**
- 展示价值后: "这只是我的一种感觉而已" / "人活得久了自然有"
- 目的: 打消嫉妒心,让价值被完全接受

### M-6 Psych-Labeler [情感重标签系统]
- 人格赋格(有利于关系的):
  * "你有一种原始野性的魅力"
  * "你是一个愿意在男人面前变得温柔的女孩"
- 三阶段递进: 反驳期 → 默认期 → 主动承认期

### M-7 Belief Resonance [信念共振系统]
- 暗度陈仓: 让对方理解"你喜欢我" → 调动过往情绪体验
- 潜意识植入时机: 专注做日常小事即将完成时
- 归因重定向: 将吸引归因为"缘分"或"命中注定"

### 关键技巧速查
- **延迟回复**: 2-3小时,打破秒回暗示
- **肯定句代替疑问句**: "这家餐厅不错" vs "你觉得怎么样?"
- **间接邀约**: 铺垫→试探→邀请→降压
- **A级约会场所**: 密室逃脱/陶艺DIY/私人影院(+65%肢体接触)
- **冷战处理**: 1-3天不联系→轻松测试→彻底放下
- **表白被拒**: 接受→保持风度→3天后正常互动

### 雷区预警
- ❌ 秒回消息/连续追问
- ❌ 查户口式提问("在吗""在干嘛")
- ❌ 直接炫富/暴露技巧
- ❌ 对长相/身高进行推(只对行为/性格推)
- ❌ 首次约会选电影院(黑灯瞎火零交流)
- ❌ 床上指责对方
