# 输出模板

## 模板使用说明

本文档提供所有标准化的输出格式模板。
使用时，根据具体场景选择合适的模板，填充内容。

---

## 模板1：学习进度报告

**使用场景**：展示龙虾自主学习进度

```markdown
## 📊 龙虾学习进度报告

**更新时间**：[YYYY-MM-DD HH:mm]

### 整体进度
**当前阶段**：第X阶段 - [阶段名称]
**完成度**：X/Y篇（X%）
**预计完成**：[估计时间]

### 本周期学习
**新完成**：
- ✅ [编号] [标题] - [一句话核心收获]
- ✅ [编号] [标题] - [一句话核心收获]

**正在学习**：
- 📖 [编号] [标题] - [当前学习重点]

### 能力升级
**新增能力**：
- [能力描述]（来源：[文章编号+标题]）

**行为改变**：
- 之前：[旧行为描述]
- 现在：[新行为描述]

**实战应用**：
- 在[具体场景]应用了[知识点]
- 效果：[具体描述]

### 学习笔记精选
**[文章标题]**：
- 核心概念：[概念1]、[概念2]
- 方法论：[关键步骤]
- 我的准则：✅ [应该]、❌ [不应该]

### 下一步计划
- 📋 待学习：[编号] [标题]
- 🎯 重点：[学习目标]
- 💡 思考：[待深入的问题]

---

**报告生成时间**：[时间戳]
**下次更新**：[时间]
```

## 模板使用指南

### 如何选择合适的模板？

| 场景 | 使用模板 |
|------|---------|
| 展示学习进度 | 模板1：学习进度报告 |
| 讲解概念 | 模板2：概念教学 |
| 讲解方法 | 模板3：方法教学 |
| 讲解逻辑 | 模板4：逻辑教学 |
| 定制方案 | 模板5：实战方案 |
| 引导思考 | 模板6：苏格拉底引导 |
| 记录学习笔记 | 模板7：学习笔记 |
| 诊断问题 | 模板8：技能诊断 |
| 解决问题 | 模板9：问题解决方案 |
| 学习建议 | 模板10：学习建议 |
| 定期汇报 | 模板11：心跳汇报 |
| 推荐 Skill | 模板12：技能推荐 |

### 模板定制原则

1. **保持结构** - 不要删除核心结构
2. **灵活填充** - 根据实际情况填充内容
3. **删减可选** - 不相关的部分可以删除
4. **保持清晰** - 使用简洁明了的语言
5. **用户友好** - 便于用户理解和执行
